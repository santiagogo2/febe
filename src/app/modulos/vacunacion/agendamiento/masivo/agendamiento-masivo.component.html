  • tipoDocumento: Tipo de documento
  • numeroDocumento: Número de documento
  • primerNombre: Primer nombre
  • segundoNombre: Segundo nombre
  • primerApellido: Primer apellido
  • segundoApellido: Segundo apellido
  • servicio: Servicio
  • aseguradora: Aseguradora - Debe ingresar el ID de la aseguradora
  • celular: Celular
  • sedeVacunacion: Sede de vacunación
  • fechaVacunacion: Fecha de vacunación
  • horaVacunacion: Hora de vacunación
  • mesa: Mesa
  • dosis: Dosis de la Vacuna suministrada. Si es la primera dósis debe poner el número 1 si es la segunda debe poner el número 2
  • tipoUsuario: Tipo de Usuario Agendado. Para colaborador poner el número 1 si es usuario debe poner el número 2
